# Torn - Travel Profit

A Tampermonkey userscript that highlights the **3 most profitable in-stock items** when you're abroad in Torn. No clicking — it just glows the good stuff so you can grab it fast.

## Prerequisites
- Torn Tools

## What It Does

When you land in a foreign country the script reads the profit figures already on the page (from Torn Tools) and ranks what's worth buying. The top three in-stock items get a colored border:

- **Green** — best profit
- **Orange** — second best
- **Red** — third best

Items with zero stock are ignored. If you're not abroad, it does nothing.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser
2. Open Tampermonkey → **Create a new script**
3. Paste in `torn-travel-profit.user.js` and save

Or install from Greasy Fork 

## Usage

Just travel. Open a foreign shop like you normally would — the highlights show up automatically. Buy what you want; the script doesn't place orders or change prices.
